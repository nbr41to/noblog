## Notion Blog の運営

大好きな Notion に囲まれて 1 年半が経つだろうか。
Notion とは一言で言えば万能メモアプリだ。
このアプリでメモを取るのが本当に楽しい。
その楽しさからか出会って 2 日でショートカットキーをほぼ全て覚えてしまった。
さらに私の学ぶ React/TypeScript との相性が非常によく、最近公開された Notion API（Beta 版） を用いて Blog の投稿・作成が快適にできるようになった。
Notion 内のデータベースを Blog の記事にできるのである。
そして私は 1 年の時を経て、それらを利用してこの Web サイトを作成することで、エンジニアとしての成長を実感することもできている。
これらの想いは熱く語らずにはいられない。

## フロントエンドエンジニア

主に React ライブラリを使った準委任契約でのフロントエンド受託開発を担っている。
Next.js と TypeScript を用いて開発することが多い。
その他には Gatsby.js, Liff, Firebase, Storybook, HeadressCMS などを用いた開発経験もある。
最近は GraphQL API や Node.js や Go 言語でのバックエンドの勉強をしている。とにかく新しい技術を学んだり使ったりすることが好き。
ガジェットとともに新しい IT 関係の知識をキャッチアップする好奇心がある。

## progL 運営

### progLearning

学習コンテンツ。

自分の学習のアウトプットと無料教材の提供を同時に行う。
主に [YouTube チャンネル](https://www.youtube.com/channel/UCPcjWvYIfvqGPP4x30kEkMA)での動画を投稿や、学習内容を Notion でまとめたりしている。

### progLab

学習コミュニティ。

学ぶことの楽しさを共有したいことから、プログラミングを学ぶ人たちのコミュニティを設立した。
現在は 10 名以上のメンバーが集まっており、定期的なイベントや共同開発を頻繁に実施している。
フロントエンドや React を中心としたプログラミングの知識を内部でも発信中。

###### [もっと詳しく](https://proglab.vercel.app/)
