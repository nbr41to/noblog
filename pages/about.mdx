# noblog について

このブログもしくは,私に興味を持っていただきありがとうございます！
このブログは NotionAPI（PublicBETA 版）と Next.js を使って作成されています.
残念ながら,簡単に作成することはできません.私ができることは,作成代行（有料）と作成に必要な知見を教えることです.
興味があればご連絡をお待ちしております.

## Notion について

メモアプリ.

## NotionAPI について

Notion に書いた内容を呼び出すことができます.

## Next.js について

JavaScript のライブラリである React のフレームワークでブログサイトに適した機能が盛りだくさん.

## その他に使用している技術

- Google Analytics
- Google AdSense
- styled-components
- MDX
- eslint
- prettier
